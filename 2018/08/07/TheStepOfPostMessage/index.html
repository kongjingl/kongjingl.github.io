发布新文章的步骤 ：（我的博客安装目录为：F:\blog）
0、在线通过网址http://marxi.co/记录文字内容，生成md格式的文件并放到目录：F:\blog\source\_posts
1、通过 Git Bash进入相关目录：cd F:\blog
2、清除缓存：hexo clean
3、根据Port的内容，生成文件：hexo g
4、部署到服务器：hexo d
5、启动服务器：hexo s
